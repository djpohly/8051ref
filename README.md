# 8051 Quick Reference

This is a quick reference sheet/cheatsheet for the 8051 microcontroller
architecture.  It includes a compact list of all assembly instructions with
their opcodes, number of cycles, and affected flags, as well as the memory
layout and special registers of the 8051.

The original was created by Willem van Engen and is available [from his
website](http://willem.engen.nl/uni/elec/).
